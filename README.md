O conteúdo deste repositório corresponde a um projeto desenvolvido pelo Grupo 26 na disciplina de Enterprise Data Science Bootcamp no âmbito da Pós-Graduação de Enterprise Data Science and Analytics da Nova IMS em 2025.

Este projeto tem como finalidade compreender os fatores que influenciam a rotatividade de colaboradores numa organização, através de uma análise exploratória detalhada e da identificação das variáveis-chave que mais contribuem para esse efeito. A partir dessa base, foi desenvolvido um modelo preditivo de machine learning capaz de antecipar a probabilidade de saída de cada colaborador. Os insights gerados serviram de suporte a recomendações estratégicas e ações direcionadas, com o objetivo de aumentar a retenção de talento e fortalecer a competitividade da empresa.

O repositório está organizado da seguinte forma: 
- EDSB25_26.csv (inclui a identificação do grupo e de cada elemento)
- Pasta src (contém o ficheiro hr_attrition.py com o código necessário para este projeto)
- .gitignore (ficheiro que descreve o conteúdo que deve ser ignorado pelo git)
- HR Attrition Dataset Variables.pdf (documento PDF com o descritivo das variáveis que existem no dataset)
- Predicting Employee Attrition.pdf (documento PDF com o enunciado do projeto)
- README.md (presente ficheiro)
- requirements.txt (ficheiro txt que detalha todas as bibliotecas e pacotes necessários para que o projeto funcione corretamente)
