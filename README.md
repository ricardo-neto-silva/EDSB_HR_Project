O conteúdo deste repositório representa um projeto desenvolvido pelo Grupo 26 na disciplina de Enterprise Data Science Bootcamp no âmbito da Pós-Graduação de Enterprise Data Science and Analytics da Nova IMS em 2025.

Este projeto tem como finalidade compreender os fatores que influenciam a rotatividade de colaboradores numa organização, através de uma análise exploratória detalhada e da identificação das variáveis-chave que mais contribuem para este efeito. A partir dessa base, será desenvolvido um modelo preditivo de machine learning capaz de antecipar a probabilidade de saída de cada colaborador. Os insights gerados servirão de suporte a recomendações estratégicas e ações direcionadas, com o objetivo de aumentar a retenção de talento e fortalecer a competitividade da organização.
